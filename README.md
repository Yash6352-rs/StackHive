# 🚀 StackHive

A modern internal knowledge-sharing and Q&A platform built with Flutter and Firebase, designed to help teams collaborate, ask questions, share expertise, and discover solutions efficiently.

StackHive enables users to create questions, provide answers, vote on helpful responses, bookmark important discussions, receive notifications, and leverage AI-generated answers. The platform also includes a powerful admin panel for moderation, analytics, user management, and content governance.

## 🏗️ Architecture

StackHive follows a scalable and maintainable architecture using:

- Feature-Based Folder Structure (MVVM style)
- Riverpod State Management
- Repository Pattern
- Firebase Backend Services
- GoRouter Navigation
- Clean Separation of Concerns
